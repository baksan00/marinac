# Secure Chat called Marinac

End-to-end encrypted real-time chat application for cryptography project.

Designed for Winows OS and Linux OS !

It is chat-communication platfrom living in your temporary terminal environment for Enterprise SECRET peer-to-peer comunication. :D

## Features
- Secure authentication
- E2EE messaging
- X25519 key exchange
- AEAD encryption
- Replay protection

## Requirements
- Python 3.11+
- Git

## Client install (Linux)
```bash
git clone https://github.com/ime/secure-chat.git
cd secure-chat
bash scripts/install-client.sh
bash scripts/run-client.sh
```

## Client install (Windows)
```bash
git clone https://github.com/ime/secure-chat.git
cd secure-chat
bash scripts/install-client.sh
bash scripts/run-client.sh
```
