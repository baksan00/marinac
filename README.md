# Secure Chat called Marinac

End-to-end encrypted real-time chat application for a university cryptography project.

Designed for Winows OS and Linux OS !

It is one-time chat-communication platfrom in real private environment

## Features
- Secure authentication
- E2EE messaging
- X25519 key exchange
- AEAD encryption
- Replay protection

## Requirements
- Python 3.11+
- Git
